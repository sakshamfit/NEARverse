# NEARverse Realistic Models

## How to Add Realistic Characters

### Step 1: Create Characters
1. Go to https://readyplayer.me
2. Create realistic male and female avatars
3. Download as **FBX**

### Step 2: Add Animations (Mixamo)
1. Go to https://www.mixamo.com
2. Upload your character
3. Download these animations:
   - Idle
   - Walking  
   - Running
   - Wave
   - Sitting

### Step 3: Convert to GLB
1. Open the FBX in **Blender**
2. Export as **GLB** (Binary + Embed Textures)

### Required Model Files

Place these files in this folder:

- `male-realistic.glb` (Default male)
- `female-realistic.glb` (Default female)
- `doctor-realistic.glb`
- `chef-realistic.glb`
- `developer-realistic.glb`
- `worker-realistic.glb`
- `photographer-realistic.glb`
- `designer-realistic.glb`

### Quick Start (For Testing)

You can use any GLB character model for testing.
The system will automatically use `male-realistic.glb` as fallback.

### Recommended Free Sources

- Ready Player Me (https://readyplayer.me)
- Mixamo (https://www.mixamo.com)
- Sketchfab (search "realistic human glb")
- Poly Pizza (free low-poly to realistic models)
