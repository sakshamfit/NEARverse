## Overview

NEARverse is a real-time multiplayer 3D social world. It uses a React frontend with Three.js for rendering the 3D environment, and a Node.js/Express backend with Socket.IO for real-time multiplayer synchronization. It integrates with Supabase for authentication and database management.

## Data Flow

1. User authenticates via Supabase on the frontend.
2. The authenticated user joins the game, initializing a Socket.IO connection to the backend.
3. The frontend continuously sends player position/rotation to the backend.
4. The backend broadcasts player positions and chat messages to nearby connected clients (Proximity Chat).
5. Zustand manages the global state (current user, nearby players, UI state) on the frontend.
